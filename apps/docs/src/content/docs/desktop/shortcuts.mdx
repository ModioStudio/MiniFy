# Keyboard Shortcuts

MiniFy is designed to be controlled entirely with your keyboard. Learn these shortcuts to maximize your productivity.

## Default Shortcuts

### Playback Control

<Tabs>
  <TabItem label="Windows/Linux">
    | Action | Shortcut |
    |--------|----------|
    | Play / Pause | `Space` |
    | Next Track | `→` or `N` |
    | Previous Track | `←` or `P` |
    | Seek Forward | `Shift + →` |
    | Seek Backward | `Shift + ←` |
  </TabItem>
  <TabItem label="macOS">
    | Action | Shortcut |
    |--------|----------|
    | Play / Pause | `Space` |
    | Next Track | `→` or `N` |
    | Previous Track | `←` or `P` |
    | Seek Forward | `Shift + →` |
    | Seek Backward | `Shift + ←` |
  </TabItem>
</Tabs>

### Volume Control

| Action | Shortcut |
|--------|----------|
| Volume Up | `↑` |
| Volume Down | `↓` |
| Mute / Unmute | `M` |

### Window Control

<Tabs>
  <TabItem label="Windows/Linux">

    | Toggle Always on Top | `Ctrl + T` |

  </TabItem>
  <TabItem label="macOS">

## Global Shortcuts

Global shortcuts work even when MiniFy is in the background or minimized.

<Aside type="caution">
  Global shortcuts may conflict with other applications. You can customize them in Settings.
</Aside>

### Default Global Shortcuts

| Action | Windows/Linux | macOS |
|--------|---------------|-------|
| Show MiniFy | `Alt + M` | `⌥ + M` |
| Play / Pause | `Alt + Space` | `⌥ + Space` |
| Next Track | `Alt + →` | `⌥ + →` |
| Previous Track | `Alt + ←` | `⌥ + ←` |

## Customizing Shortcuts

You can customize any shortcut to fit your workflow:

1. Open Settings (`Ctrl+,` or `⌘+,`)
2. Navigate to "Keyboard Shortcuts"
3. Click on the shortcut you want to change
4. Press your new key combination
5. Click "Save"

### Supported Modifiers

- **Windows/Linux**: `Ctrl`, `Alt`, `Shift`, `Win`
- **macOS**: `⌘`, `⌥`, `⌃`, `⇧`

### Supported Keys

Most standard keys are supported:
- Letters: `A` - `Z`
- Numbers: `0` - `9`
- Function keys: `F1` - `F12`
- Arrow keys: `←`, `→`, `↑`, `↓`
- Special keys: `Space`, `Enter`, `Tab`, `Escape`

## Resetting Shortcuts

To reset all shortcuts to their defaults:

1. Open Settings
2. Navigate to "Keyboard Shortcuts"
3. Click "Reset to Defaults"

<Aside type="tip">
  Create a "cheat sheet" by exporting your shortcuts configuration from Settings → Export.
</Aside>
