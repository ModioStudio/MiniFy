# Quick Start

This guide will help you set up MiniFy and start controlling your Spotify playback in under 5 minutes.

## Prerequisites

Before you begin, make sure you have:

- âœ… MiniFy installed on your system ([Installation Guide](/guides/installation/))
- âœ… A Spotify account (Free or Premium)
- âœ… Spotify playing on some device (phone, computer, smart speaker, etc.)

<Aside type="tip">
MiniFy is a **controller** - it needs an active Spotify playback session to control.
Start playing something in Spotify first, then MiniFy can display and control it.
</Aside>

## Customization Quick Guide

### Change Theme

1. Right-click anywhere in MiniFy
2. Hover over "Themes"
3. Select your preferred theme

**Available themes:** Dark, Light, Catppuccin, Dracula, ChatGPT, YouTube, BMW, Milka

### Change Layout

1. Right-click anywhere in MiniFy
2. Hover over "Layout"
3. Choose from:
   - **Layout A** - Compact (minimal footprint)
   - **Layout B** - Standard (balanced view)
   - **Layout C** - Expanded (large album art)

### Enable Always on Top

1. Right-click anywhere in MiniFy
2. Click "Always on Top"

This keeps MiniFy visible above other windows.

## Setting Up AI DJ (Optional)

The AI DJ gives you personalized music recommendations. To set it up:

<Steps>
1. **Open Settings**

   Press `Ctrl+,` (or `âŒ˜+,` on macOS), or right-click â†’ Settings

2. **Go to Connections**

   Navigate to the "Connections" tab

3. **Add an AI Provider**

   Choose from OpenAI, Anthropic, Google AI, or Groq and enter your API key

4. **Activate the Provider**

   Click "Connect" to validate, then set it as **Active**

5. **Use AI DJ**

   Right-click â†’ AI DJ to open the chat interface
</Steps>

## Troubleshooting First Setup

### MiniFy won't connect to Spotify

- Ensure you have an internet connection
- Check that your browser can access spotify.com
- Try clearing browser cookies for Spotify

### No track is showing

- Make sure Spotify is playing on some device
- Check that the same Spotify account is used
- Try playing/pausing in Spotify to refresh the connection

### Controls don't work

- Verify you have an active Spotify playback device
- Premium features (like seeking) may be limited on Free accounts
- Try clicking play in Spotify first
