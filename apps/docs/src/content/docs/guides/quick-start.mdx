# Quick Start

This guide will help you set up MiniFy and start controlling your Spotify playback in under 5 minutes.

## Prerequisites

Before you begin, make sure you have:

- ✅ MiniFy installed on your system
- ✅ A Spotify account (Free or Premium)
- ✅ The Spotify desktop app or web player running

## Getting Started

<Steps>
1. **Launch MiniFy**
   
   Open MiniFy from your applications menu or desktop shortcut.

2. **Connect to Spotify**
   
   On first launch, you'll see a "Connect to Spotify" button. Click it to begin the authentication process.

3. **Authorize access**
   
   A browser window will open asking you to log in to Spotify and authorize MiniFy. Click "Agree" to continue.

4. **Start playing music**
   
   Once connected, MiniFy will display your currently playing track. If nothing is playing, start a song in Spotify.

5. **Explore the interface**
   
   - Click the album art to expand/collapse the player
   - Use the playback buttons to control music
   - Right-click anywhere for the context menu
</Steps>

## Essential Shortcuts

Learn these shortcuts to control MiniFy like a pro:

| Action | Windows/Linux | macOS |
|--------|---------------|-------|
| Play/Pause | `Space` | `Space` |
| Next Track | `→` | `→` |
| Previous Track | `←` | `←` |
| Volume Up | `↑` | `↑` |
| Volume Down | `↓` | `↓` |
| Open Settings | `Ctrl+,` | `⌘+,` |

<Aside type="tip">
  You can customize these shortcuts in Settings → Keyboard Shortcuts.
</Aside>

## Customizing Your Experience

### Change Theme

1. Right-click anywhere in MiniFy
2. Select "Themes" from the context menu
3. Choose your preferred theme

### Change Layout

1. Right-click anywhere in MiniFy
2. Select "Layout" from the context menu
3. Choose between Compact, Standard, or Expanded

### Adjust Settings

Press `Ctrl+,` (or `⌘+,` on macOS) to open Settings where you can:

- Enable/disable "Always on top"
- Customize keyboard shortcuts
- Adjust transparency
- And more...
