## Technology Stack

MiniFy is built with modern, efficient technologies:

| Component | Technology | Purpose |
|-----------|------------|---------|
| Runtime | Tauri 2.0 | Native desktop shell |
| Backend | Rust | Secure operations, OAuth |
| Frontend | React 18 | User interface |
| Build | Vite | Fast bundling |
| AI | Vercel AI SDK | Multi-provider AI support |
| Styling | CSS Variables | Dynamic theming |
